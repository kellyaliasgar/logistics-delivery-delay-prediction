# Logistics Delivery Delay Prediction

Machine learning logistics analytics project focused on predicting delivery delays using operational transportation data.

## Project Overview

This project simulates a logistics delivery environment and applies machine learning techniques to predict whether a delivery will be delayed based on operational factors such as:

- traffic conditions
- weather conditions
- delivery distance
- package weight
- delivery window
- driver experience
- weekend operations

The project demonstrates a complete machine learning workflow including:

- exploratory data analysis (EDA)
- correlation analysis
- multicollinearity testing (VIF)
- outlier analysis
- feature engineering
- preprocessing pipelines
- supervised machine learning
- cross-validation
- feature importance analysis
- logistic regression interpretation
- business-oriented operational insights

---

## Machine Learning Models Used

- Logistic Regression
- Random Forest Classifier
- Gradient Boosting Classifier

---

## Evaluation Metrics

The models were evaluated using:

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC
- Cross-Validation Performance

---

## Key Findings

Key operational findings from the analysis included:

- Longer delivery distances significantly increased delay probability
- Weekend deliveries showed higher operational delay risk
- Driver experience reduced delivery delay likelihood
- Traffic and weather conditions were among the strongest predictors of delays
- Ensemble models effectively captured nonlinear operational relationships
